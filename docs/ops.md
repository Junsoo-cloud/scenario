# 운영/보안 체크리스트 (초기 MVP)

## 보안
- 비밀번호 해시: bcrypt 적용
- JWT 만료 2h, refresh 토큰 14d
- 파일 다운로드 URL은 서명 기반 (S3 presigned URL)
- 업로드 파일 확장자/크기 제한 + 바이러스 스캔 큐 연동
- 신고/제재 이력 감사 로그 기록

## 운영
- 로그: API 요청/에러 로그 JSON 포맷
- 모니터링: 5xx 비율, 응답시간, 결제 실패율
- 알림: 결제 실패, 검수 지연, 신고 급증

## 백업/복구
- DB 일 1회 스냅샷, 7일 보관
- 스토리지 버전 관리 ON

## 법적
- 이용약관/개인정보/저작권 안내 상시 노출
- 환불 정책 명문화
